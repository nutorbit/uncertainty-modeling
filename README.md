# Uncertainty modeling

## Development setup

This repository uses [Poetry](https://python-poetry.org/) as a dependency manager. To install the dependencies, run:

```zsh
$ poetry install
```

To run the jupyter notebook:

```zsh
$ poetry run jupyter lab
```
